E-commerce website to add products to cart.

Functionalities:
Add to cart button: Adds items to the cart array as well as the cart list for rendering
Checkout button: Provides "checkout successful" message upon clicking
Summation: With each click of the add to cart button the cart total is updated with the summation of the prices of the products in the cart
Empty cart message: Show "cart is empty" message when no products are included in the cart
